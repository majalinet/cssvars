Responsive Embeds

To make responsive embeds:
  • Create tow divs
    • Outer div with class embed and size class, like embed-wide-screen
    • Inner div with class embed-item
  • Classes for embed sizes:
    • embed-ultra-wide - 21:9 aspect ratio
    • embed-wide-screen - 16:9 aspect ratio
    • embed-standard: - 4:3 aspect ratio
    • embed-square - 1:1 aspect ratio
  • Make sure to remove padding on the outer div and parent elements
Responsive embed-ultra-wide

ON div class="embed embed-ultra-wide"
on iframe class="embed-item"

Responsive embed-wide-screen

ON div class="embed embed-wide-screen"
on iframe class="embed-item"

Responsive embed-standard

on div class="embed embed-standard"
on iframe class="embed-item"

Responsive embed-square

on div class="embed embed-square"
on iframe class="embed-item"
